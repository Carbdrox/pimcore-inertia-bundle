Chapter 1: Introduction and Overview

1.1 What is the Pimcore Inertia Bundle?

The Pimcore Inertia Bundle is an innovative extension for the Pimcore Content Management System that enables seamless integration between Pimcore and the JavaScript framework Inertia.js. This integration allows developers to create modern Single-Page-Applications (SPAs) with JavaScript frameworks like Vue.js directly within Pimcore, without losing the benefits of server-side architecture.

1.2 The Challenge

Traditionally, developers face a choice when building web applications between:

  • Server-side applications with full page reloads
  • Client-side SPAs with complex API development and state management

The Pimcore Inertia Bundle solves this dilemma by combining the best of both worlds.

1.3 The Solution: Inertia.js Integration

Inertia.js acts as a bridge between modern JavaScript frameworks and classic server-side frameworks. With the Pimcore Inertia Bundle, you can:

  • Use Vue.js components directly in Pimcore
  • Maintain server-side routing and controllers
  • No API development required
  • Progressive enhancement for existing Pimcore projects
  • Create SEO-friendly applications

1.4 Architecture Overview

graph LR A[Pimcore Controller] --> B[Inertia Response] B --> C[JavaScript Framework] C --> D[Vue.js Component] D --> E[User Interface] F[Pimcore Models] --> A G[Pimcore Services] --> A

The bundle works as follows:

  1. Pimcore Controllers process requests as usual
  2. Inertia Response sends JSON data instead of HTML views
  3. JavaScript Framework (Vue.js) receives the data
  4. Components render the user interface client-side
